# Environment Setup

## Target Application
OWASP Juice Shop

## Setup Summary
For this project, OWASP Juice Shop was run locally using Docker.
The application was started as a local container and exposed on
port 3000, allowing access through a web browser at localhost.
Running the application locally ensured all testing was performed
in a safe and controlled lab environment.


## Deployment Method
Docker (local container-based deployment)

## Access URL
http://localhost:3000

## Notes
The application was accessed through a web browser after deployment.
No external or real-world systems were involved in testing.
